# react-magic-mirror
An API driven magic mirror project that will interface with devices such as Alexa & Google Maps.

## Features

* WIP - Weather
* WIP - Traffic
* Greeting
* Date & Time

Only tested on Google Chrome

## Running this app
`npm start`

## Build
`gulp build`

## Dev ( Watch + Build )
`gulp dev`

## Uses the following API's

[Dark Sky](https://darksky.net)
